## Running locally
**Requirements:**

- **Typescript Knowledge**
- [Node.JS](https://git-scm.com/)
- [Git](https://nodejs.org/en)

**Steps:**
```bash
git clone https://github.com/0xQueen/homepage/
cd homepage
npm i
npm run dev
```

To open the website, visit [http://127.0.0.1:3000](http://127.0.0.1:3000).
